# Advanced Guess The Number Game

## Project Description

Advanced Guess The Number Game is a Python-based interactive console application where users attempt to guess a randomly generated number. The game includes multiple difficulty levels, scoring mechanisms, high score tracking, achievement badges, and game history storage.

---

## Features

* Random Number Generation
* Easy, Medium and Hard Difficulty Levels
* Player Name Registration
* Limited Chances Based on Difficulty
* Hint System (Even/Odd Hint)
* High Score Tracking
* Timer-Based Scoring
* Achievement Badges
* Game History Storage
* Input Validation
* Play Again Option

---

## Difficulty Levels

| Level  | Range   | Chances |
| ------ | ------- | ------- |
| Easy   | 1 - 50  | 10      |
| Medium | 1 - 100 | 8       |
| Hard   | 1 - 500 | 6       |
